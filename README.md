# Basic Chatbot

## Description
This is a simple rule-based chatbot built using Python. The chatbot can respond to basic greetings and questions using predefined replies. It demonstrates fundamental programming concepts like **functions, if-elif statements, loops, and input/output**.

## Features
- Responds to greetings like "hello" or "hi".
- Responds to questions like "how are you".
- Says goodbye when the user types "bye" or "exit".
- Handles unknown inputs with a default response.

## How to Run
1. Make sure you have **Python installed** (Python 3.x recommended).
2. Clone or download this repository.
3. Open a terminal/command prompt and navigate to the project folder.
4. Run the chatbot with:
   python chatbot.py
5.Interact with the chatbot by typing messages. Type "bye" to exit.

Example Interaction
Welcome to the Chatbot! (type 'bye' to exit)
You: hello
Bot: Hi there!
You: how are you
Bot: I'm fine, thanks! How about you?
You: bye
Bot: Goodbye! Have a nice day!

Key Concepts Used

Functions: To handle responses.

if-elif statements: For matching user input.

Loops: To continuously interact with the user.

Input/Output: To read user input and display bot responses.
